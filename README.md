# Spring Boot Teacher Student Crud

## Models and Database
- Multi user login
- Crud (add, update, delete)
- Postgres SQL / MySQL

## Run app
### Clone project
- git clone https://github.com/Ochibooh/spring-boot-student-teacher-crud.git
### Compile project
1. mvn clean
2. mvn clean install
3. Go to the target folder
4. java -jar student_teacher_crud-0.0.1.war
